4 网站创建

4.1 Jeklle

4.1.1 installation

参考Github+Jekyll 搭建个人网站详细教程

  • Ruby installer

    //进入到解压包的位置
    ruby setup.rb

  • RubyGems

    gem install jekyll

  • 创建test blog,并启动服务器

    jekyll new testblog
cd testblog
jekyll server
在浏览器输入http://127.0.0.1:4000/即可浏览刚刚创建的blog

  • jekyll 主题官网

    [adam-blog](adam-blog)
命令行进入该目录执行jekyll server

    若下载的主题jekyll server执行失败，则用步骤二中创建的testblog目录下的Gemfile，Gemfile.lock文件替换下载的主题里面的该文件，若还不成功，则根据控制台提示的错误，可以百度到解决方案。
